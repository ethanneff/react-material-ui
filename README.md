# React Boilerplate

React + Redux Toolkit + TypeScript + React Router + Material UI

### Install

```sh
yarn install
```

### Develop

```sh
yarn start
```

### Contribute

```sh
yarn lint
```

```sh
yarn test
```

### Deploy

```sh
yarn build
```
